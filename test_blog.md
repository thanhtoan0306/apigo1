# Hướng dẫn Golang cơ bản

## Giới thiệu

Golang (hay Go) là một ngôn ngữ lập trình được phát triển bởi Google. Đây là ngôn ngữ hiện đại, hiệu suất cao và dễ học.

## Tính năng chính

- **Đơn giản**: Cú pháp dễ hiểu, dễ học
- **Hiệu suất cao**: Compile sang machine code, chạy nhanh
- **Concurrent**: Hỗ trợ goroutines và channels
- **Cross-platform**: Chạy trên nhiều hệ điều hành

## Ví dụ code

```go
package main

import "fmt"

func main() {
    fmt.Println("Hello, World!")
}
```

## Kết luận

Golang là lựa chọn tuyệt vời cho backend development!
